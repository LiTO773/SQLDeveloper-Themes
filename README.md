# SQLDeveloper Themes

## Intro
SQLDeveloper Themes is a collection of color schemes and UI themes for Oracle's 
SQLDeveloper. A JavaFX installer is also provided, since it is pretty cumbersome
to install this themes manually.

## Color Schemes
